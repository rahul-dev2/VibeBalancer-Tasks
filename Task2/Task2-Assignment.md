## This assignment consists of **three problems** to test your understanding of Python fundamentals.



## Question 1 – File Handling

Write a Python program that:

1. Creates a text file named `data.txt`.
2. Writes at least **5 lines of text** into the file.
3. Reads the contents of the file.
4. Displays the contents on the console.


---

## Question 2 – Problem Solving (Python Basics)

Write a Python program that:

- Takes a list of numbers as input.
- Finds and prints:
  - The **largest number**
  - The **smallest number**
  - The **sum of all numbers**
  - The **sum of squares of all numbers**
  - The **prime numbers in the list**

### Example

Input: [4, 7, 1, 9, 3] 

Output:
- Largest: 9
- Smallest: 1
- Sum: 24
- Sum of sqaures: 156
- Primes: [3,7]

## Question 3 – Object Oriented Programming (OOP)

Create a class called **Student** with the following:

### Attributes

- `name`
- `roll_number`
- `marks`

### Methods

- A method to **display student details**
- A method to **check if the student has passed**  
  (Assume passing marks = 40)

### Example Output
- Name: Rahul
- Roll Number: 12
- Marks: 65
- Status: Passed
