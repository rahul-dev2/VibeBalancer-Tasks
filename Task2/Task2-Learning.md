### What is Python?

Python is a **high-level, beginner-friendly programming language** known for its simple syntax and powerful capabilities.

During this learning phase, your objective is to understand the basic concepts of Python such as:

- Variables and data types
- Conditional statements
- Loops
- Functions
- Basic input and output
- Introduction to file handling
- Introduction to Object-Oriented Programming (OOP)

Make sure you **install Python on your system** and try running small programs while learning.
